## Description
Show current C2 configuration including callback host, callback port, sleep interval, jitter, and killdate.

## Usage
```
c2info
```

### Examples
```
c2info
```

### Output
Displays:
- Callback Host
- Callback Port
- Sleep Interval (seconds)
- Jitter (%)
- Kill Date (YYYY-MM-DD)

None (operational command)
