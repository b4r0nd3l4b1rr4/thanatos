## Description
Revert to the agent's original token (RevertToSelf).

### Parameters
None.

## Usage
```
token_revert
```

## Notes
 - Stops impersonating any previously applied token
 - Agent returns to its original security context

## MITRE ATT&CK Mapping
 - T1134
